

Перший абстрактний слой.

Image - це типу класи, які мають приватні поля, публічні поля і наслідування.
	Характеристики - це приватні поля
	Інградієнти - це наслідування
	Ссилка це розширення за допомогою класу (Тобто поле Фірма ссилається на клас -> Фірма 
	в якому розширено описані властивості цього поля. Це свого роду як типу у C#)

Наприклад: клас Ноутбук

класс Абстрктні поняття
{
	Ноутбук -> класс Ноутбук
	Товар -> Товар
	Магазин -> Магазин
}

класс Ноутбук
{
	Назва
	Процесор -> класс Процесор
	Монітор -> класс Монітор
	ПО -> ПО
	Фірма -> класс Фірма
}

класс Процесор
{
	Назва
	Фірма -> класс Фірма
}

класс Монітор
{
	Назва
	Фірма -> класс Фірма
}

класс Фірма
{
	Назва
}

класс ПО
{
	Назва
}

класс Товар
{
	Назва
	Ціна
}

класс Магазин 
{
	Назва
	Товари -> Товар
}

Другий абстрактний слой.

Pictures - це слой в якому відбувається обєднання Images і Pictures, їхнє злиття і утворення нових
структур.

Наприклад: 

Ноутбук -> Ноутбук <клас Абстрктні поняття> - створюємо новий малюнок Ноутбук який вказує на ноутбук в слої 1.
Товар -> Товар <клас Абстрктні поняття> - створюємо новий малюнок Товар який вказує на товар в слої 1.
Магазин -> Магазин <клас Абстрктні поняття> - створюємо новий малюнок Магазин який вказує на Магазин в слої 1.

Ноутбук+Товар
{
	Назва
	Ціна
	Ноутбук ->  Ноутбук
}

Тобто утворилась нова структура типу Товар яка включає в себе Ноутбук і тепер цю структуру можна наприклад
обєднати і структурою Магазин
 
Це обєднання поміщає Ноутбук+Товар в поле Товари класу Магазин
Магазин+Ноутбук+Товар
{
	Назва
	Товари 
	{
		Ноутбук+Товар
	}
}

Це вже зовсім інший малюнок який доповнює загальну картину образу Магазин. 
Тобто в Магазині є ноутбук і це не товар а наприклад комп для обліку товарів 
Магазин+Ноутбук
{
	Назва
	Товари
	Ноутбук ->  Ноутбук
}
